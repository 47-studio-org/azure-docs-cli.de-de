1. Führen Sie den Anmeldebefehl aus.

    ```azurecli-interactive
    az login
    ```

   Sie erhalten einen Code, den Sie im nächsten Schritt benötigen.

2. Verwenden Sie zum Öffnen der Seite [https://aka.ms/devicelogin](https://aka.ms/devicelogin) einen Webbrowser, und geben Sie dann zur Authentifizierung den Code ein.

3. Melden Sie sich im Browser mit Ihren Anmeldeinformationen an.
